This folder contains the test data.

"* test scenes" folders contain the required appsettings for building the scene
test images - the used images in the thesis
frame profiles - some proof images for the RQ-Count and other data
TestResults.xlsx contains the test results as well as data and required reconstruction information

For reconstructing the scenes with the appsettings see the SceneCompiler project

DISCLAIMER
When reproducing the tests, remember window resolution and turn of vsync, opaque check and reload the shader
